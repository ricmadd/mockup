WBS - Work Breakdown Structure

The WBS summarizes all the activities and tasks that must completed on a project. As a user I would like to be able to create several different lists and organize these hierarchycally with several levels of subtasks. The WBS can also be used as the master template to create a Project Budget and Project Schedule (THIS SECTION IS THE MVP)

View details »

Project Budget

The Project Budget includes all project expense items at a great level of detail (usually WBS Level 7). In construction, the expense items usually include: Materials, Labor, Equipment, Subcontractors, and Others. As a user I would like to create and manage the budget in the WebApp as the project advances through time. (NOT MVP)
